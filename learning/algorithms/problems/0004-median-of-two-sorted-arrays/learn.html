4. 寻找两个正序数组的中位数

Hard Array · Binary Search · Partition 题面 analysis.md

术语：median · partition · binary search · sentinel \((-\infty,+\infty)\)

学习怎么用 教育场景

本题核心只有两样：定位不变量 + 边界用哨兵统一。 先把不变量说清楚，再谈 binary search；公式只记一组（下方 KaTeX）。 测验统一提交；全对剪贴板含 #题号 · slug · 用时。

  • 自学：本质 → 不变量 → 二分方向 → 边界 → 手推 → 测。
  • 复习：口头复述「数量钉死 \(j\) + 交叉两条」与「\(\pm\infty\) 消特判」。
  • 卡关：先看「\(i\) 是什么」和「为何在短数组上二分」。

本质：中位数 = 找合法 partition 先立靶

不要先想「怎么 merge」。已排序的两数组，中位数只要求： 把合并后的全体切成全局左袋 \(L\) 与右袋 \(R\)，使 \(L\) 中每个元素 \(\le R\) 中每个元素， 且 \(|L|=h\)。切缝找到了，median 就在切缝两侧。

\(i\) 不是「左半个数」。 \(L\) 是合并后的全局左袋，由 \(A\) 的前缀 + \(B\) 的前缀拼成。 我们只在 \(A\) 上选一个切分点（cut） \(i\)： 「\(A\) 的前 \(i\) 个进 \(L\)，后 \(m-i\) 个进 \(R\)」。 同理 \(j\) 是 \(B\) 上的 cut。口语里别说「左半个数」，说「\(A\) 贡献给 \(L\) 几个」。

只记这一组

数量：全局 \(|L|=h\)；\(i=\) \(A\) 的 cut，\(j=\) \(B\) 的 cut（由 \(i\) 钉死）

\[ h=\left\lfloor\frac{m+n+1}{2}\right\rfloor,\qquad j=h-i \]

顺序不变量（交叉两条即可）

\[ A_{i-1}\le B_{j} \;\land\; B_{j-1}\le A_{i} \]

答案

\[ \mathrm{median}= \begin{cases} \max(A_{i-1},\,B_{j-1}) & m+n\text{ 为奇数} \\[0.35em] \dfrac{\max(A_{i-1},\,B_{j-1})+\min(A_{i},\,B_{j})}{2} & m+n\text{ 为偶数} \end{cases} \]

边界：缺的一边用哨兵，公式不变

\[ A_{i-1}=\begin{cases}-\infty&i=0\\A[i-1]&i>0\end{cases} \quad A_{i}=\begin{cases}+\infty&i=m\\A[i]&i<m\end{cases} \] \[ B_{j-1}=\begin{cases}-\infty&j=0\\B[j-1]&j>0\end{cases} \quad B_{j}=\begin{cases}+\infty&j=n\\B[j]&j<n\end{cases} \]
为什么 \(h\) 里有 \(+1\)？ 把「下中位数」固定在左半：总长奇数时 \(|L|=|R|+1\)，\(\mathrm{median}=\max(L)\)； 偶数时 \(|L|=|R|\)，\(\mathrm{median}=\frac{\max(L)+\min(R)}{2}\)。 搜索过程奇偶相同，只返回值不同。

这题在现实里像什么 应用场景

  • 两本已排序成绩册，求全体中位数——不合并成第三本大表。
  • 两条有序报价带，求合并后的中位价。
  • 两路有序索引上的 order-statistic（第 \(k\) 小 / 中位数）。
  • 分布式两段有序数据的中位估计（partition 思想）。

二分不是炫技：题面要求 log 级。Merge 正确但 \(O(m+n)\)。 划分法用的是中位数的统计定义（均匀切半），把问题变成「找合法切分点」。

多种解法 · 对比 先选武器

A. Merge 双指针（基线）
Time \(O(m+n)\) Space \(O(1)\)

归并推进到中位下标。好写、好测，不满足 log 要求。

B. Binary partition（主推）
Time \(O(\log\min(m,n))\) Space \(O(1)\)

在较短数组上二分 cut \(i\)（\(A\) 贡献给 \(L\) 几个）； 数量不变量钉死 \(j=h-i\)； 用顺序不变量判合法 / 调方向。

在优化什么：不合并，只定位全局切缝。

C. 通用「第 \(k\) 小」二分
Time \(O(\log(m+n))\) Space \(O(1)\)

找第 \(k=\lfloor(m+n+1)/2\rfloor\) 与（偶数时）下一个。与划分同族，写法更「淘汰一半」。

解法换到了什么丢掉了什么
Merge直观正确log 复杂度
Partition 二分log、贴定义不变量 + 边界要记准
第 \(k\) 小可推广任意 \(k\)实现略绕

定位不变量：数量钉死 + 交叉比较 核心

1

数量不变量——全局 \(|L|=h\)，你只有一个自由度

合并后左袋一共要 \(h\) 个元素。选定 \(A\) 的 cut \(i\) 后， \(B\) 的 cut 必须是 \(j=h-i\)。 \(j\) 不是搜出来的，是算出来的。 搜索空间只剩 \(i\in[0,m]\)。

\[ L=\{\,A[0..i),\;B[0..j)\,\} \qquad R=\{\,A[i..m),\;B[j..n)\,\} \qquad |L|=i+j=h \]
2

顺序不变量——\(\max(L)\le\min(R)\)

合并后要整体有序，切缝两侧必须「左不大于右」。 因为 \(A,B\) 各自已排序，数组内部天然满足； 唯一可能破的是跨数组：

\[ A_{i-1}\le B_{j} \qquad\text{且}\qquad B_{j-1}\le A_{i} \]

等价写法（实现时常用）

\[ L_{\max}=\max(A_{i-1},\,B_{j-1}),\quad R_{\min}=\min(A_{i},\,B_{j}) \] \[ \text{valid}\iff L_{\max}\le R_{\min} \]

交叉两条 \(\Leftrightarrow\) 一个 \(L_{\max}\le R_{\min}\)。记交叉更利于写分支。

合法切分 · \(A=[1,3],\;B=[2]\) · \(h=2,\;i=1\Rightarrow j=1\)
A 1 3
B 2 +\(\infty\)
\(L=\{1,2\},\;R=\{3\}\)
交叉：\(A_{0}=1\le B_{1}=+\infty\) 且 \(B_{0}=2\le A_{1}=3\) ✓
奇数 \(\Rightarrow\) \(\mathrm{median}=L_{\max}=\max(1,2)=\mathbf{2.0}\)
偶数总长 · \(A=[1,2],\;B=[3,4]\) · \(h=2,\;i=2\Rightarrow j=0\)
A 1 2 +\(\infty\)
B −\(\infty\) 3 4
\(L=\{1,2\},\;R=\{3,4\}\)
交叉：\(2\le 3\) 且 \(-\infty\le+\infty\) ✓
偶数 \(\Rightarrow\) \(\mathrm{median}=\dfrac{L_{\max}+R_{\min}}{2}=\dfrac{2+3}{2}=\mathbf{2.5}\)

为什么在较短的数组上二分 \(i\)？ 卡点

不是「短数组更神秘」，而是：一旦定了 \(j=h-i\)，就必须始终 \(0\le j\le n\)。把 \(A\) 选成较短的那根（\(m\le n\)）， 对任意 \(i\in[0,m]\) 这个范围自动成立，实现不用额外夹逼 \(j\)。

当 \(m\le n\) 时（证明只需不等式，不必背）

\[ i=0 \Rightarrow j=h=\left\lfloor\frac{m+n+1}{2}\right\rfloor \le n \] \[ i=m \Rightarrow j=h-m=\left\lfloor\frac{m+n+1}{2}\right\rfloor-m \ge 0 \]

若反过来在长数组上搜，会出现 \(j<0\) 或 \(j>n\)，要额外 if，还容易写错。

  • 正确性：短数组二分 \(\Rightarrow\) \(j\) 合法区间天然满足。
  • 效率：搜索空间是 \(\min(m,n)\)，复杂度 \(O(\log\min(m,n))\)，贴题面。
  • 写法：开头若 \(m>n\) 就 swap 两数组（或交换角色），之后只在 \(A\) 上二分。
可以在长数组上二分，但必须自己处理 \(j\) 越界——收益小、坑更多。 工程默认：总在较短侧搜 cut。

Binary search 方向：两条违规各自单调 怎么搜

合法条件是「且」：任一交叉失败就说明 cut \(i\) 偏了，且方向确定：

看到什么含义动作
\(A_{i-1}>B_{j}\) \(A\) 贡献给 \(L\) 的偏大 \(\Rightarrow\) \(i\) 太大 缩小 \(i\)（\(\mathrm{hi}=i-1\)）
\(B_{j-1}>A_{i}\) \(A\) 贡献给 \(L\) 的偏小 \(\Rightarrow\) \(i\) 太小 增大 \(i\)（\(\mathrm{lo}=i+1\)）
两条都 \(\le\) 合法 partition 停，算 median
不要同时乱调。实现里通常先判「\(i\) 太大」，再判「\(i\) 太小」，else 即命中。

边界 case：哨兵统一，不是一堆 if 边界

Hard 点常卡在「\(A\) 不贡献给 \(L\)（\(i=0\)）/ 全贡献（\(i=m\)）/ \(B\) 同理」。 正确姿势：缺的一侧当成 \(-\infty\) 或 \(+\infty\)，交叉公式一字不改。

\[ A_{i-1}=\begin{cases}-\infty&i=0\\A[i-1]&\text{否则}\end{cases} \qquad A_{i}=\begin{cases}+\infty&i=m\\A[i]&\text{否则}\end{cases} \] \[ B_{j-1}=\begin{cases}-\infty&j=0\\B[j-1]&\text{否则}\end{cases} \qquad B_{j}=\begin{cases}+\infty&j=n\\B[j]&\text{否则}\end{cases} \]

比较与返回值仍用同一套 \(A_{i-1},A_{i},B_{j-1},B_{j}\)

\(i=0\) · \(A\) 不贡献给 \(L\)
A −\(\infty\) …
\(A_{i-1}=-\infty\Rightarrow A_{i-1}\le B_{j}\) 永真； 只靠 \(B\) 侧约束 \(i\)。
\(i=m\) · \(A\) 全部进 \(L\)
A … +\(\infty\)
\(A_{i}=+\infty\Rightarrow B_{j-1}\le A_{i}\) 永真； 另一侧同理。
  • 一数组为空 \(\Rightarrow\) 退化为单数组中位数。
  • 先保证 \(m\le n\)（必要时 swap），再只在短侧二分 cut。
  • 奇偶只改返回，不改搜索与哨兵。

手推一遍（对着不变量） 巩固

例 1 · 奇数 \(A=[1,3],\;B=[2]\)

\(h=\lfloor(2+1+1)/2\rfloor=2\)。试 \(i=1\Rightarrow j=1\)：

\[ A_{0}=1,\;A_{1}=3,\;B_{0}=2,\;B_{1}=+\infty \]

\(1\le+\infty\) 且 \(2\le 3\) → 合法。 \(\mathrm{median}=L_{\max}=\max(1,2)=2.0\)

例 2 · 偶数 \(A=[1,2],\;B=[3,4]\)

\(h=2\)；在短数组 \(A\) 上 \(i\in[0,2]\)。

试 \(i=1\Rightarrow j=1\)：\(A_{0}=1,\,A_{1}=2,\,B_{0}=3,\,B_{1}=4\)

\(1\le 4\) 但 \(3\not\le 2\) → \(B_{j-1}>A_{i}\) → \(i\) 太小

试 \(i=2\Rightarrow j=0\)：\(A_{1}=2,\,A_{2}=+\infty,\,B_{-1}=-\infty,\,B_{0}=3\)

\(2\le 3\) 且 \(-\infty\le+\infty\) → 合法。 \(\mathrm{median}=\frac{2+3}{2}=2.5\)

例 3 · 边界 \(A=[\,],\;B=[1]\)

单数组中位数 \(1.0\)（\(m=0\)，\(i\) 只能 \(0\)，\(j=h\)）。

发散：约束一变 思维

  • 不要求 log → Merge 足够，面试可先写对再优化。
  • 求第 \(k\) 小 → 同一思想，\(h\) 换成 \(k\)。
  • 三个有序数组中位数 → 多路划分更难，或换 selection 模型。

围绕不变量 + 边界。全对复制：#题号 · slug · 用时 · 下一步。

Q1. 本题最优路径的「定位目标」是？

解析：中位数 = 均匀切半后的切缝；不合并也能定位。

Q2. \(i\) 是 \(A\) 的 cut（\(A\) 贡献给全局 \(L\) 几个）。\(B\) 的 cut \(j\) 由什么决定？

解析：全局 \(|L|\) 固定为 \(h\)；\(j\) 被 \(i\) 钉死，搜索自由度只有 1。

Q3. 顺序不变量「交叉两条」是？（多选）

解析：\(A,B\) 各自有序，内部不用再判；只判跨数组交叉。

Q4. 若 \(A_{i-1}>B_{j}\)，应如何调？

解析：\(A\) 贡献给 \(L\) 的偏大，说明 cut \(i\) 太大，减小 \(i\)。

Q5. \(i=0\)（\(A\) 不贡献给 \(L\)）时，\(A_{i-1}\) 应视为？

解析：端点用 \(\pm\infty\) 哨兵，交叉比较无需额外分支语义。

Q6. \(\mathrm{nums1}=[1,2],\;\mathrm{nums2}=[3,4]\) 的 median 是？

解析：合并 \(1,2,3,4\) 偶数 \(\Rightarrow\frac{2+3}{2}=2.5\)；合法划分 \(L_{\max}=2,\,R_{\min}=3\)。

Q7. 总长为奇数时，median 取？

解析：\(h=\lfloor(m+n+1)/2\rfloor\) 把下中位数放进左半 \(\Rightarrow\) 奇数答案即 \(L_{\max}\)。
