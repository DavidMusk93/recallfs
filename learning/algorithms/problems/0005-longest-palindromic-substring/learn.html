5. 最长回文子串

Medium String · Two Pointers · DP 题面 analysis.md

术语：palindrome · substring · expand around centers · DP

学习怎么用 教育场景

先分清 substring（连续） 与 subsequence，再抓住： 回文 = 从中心对称扩展。 测验统一提交；全对剪贴板含 #题号 · slug · 用时。

  • 自学：本质 → 奇偶中心 → 扩展维护 best → 图解 → 测。
  • 复习：口头复述「\(2n-1\) 个中心」与「退出后区间是 \([L+1,R-1]\)」。
  • 卡关：偶数长度（如 cbbd）必须有「缝心」。

本质：回文由中心决定 先立靶

不要先枚举所有 \([i,j]\)。一个回文 substring 一旦中心定了，两侧能扩多远就定了。 问题变成：枚举所有可能的中心，向两侧 expand，记下最长的那一段。

只记这一组

两种中心（共 \(2n-1\) 个）

\[ \text{奇数长：中心在字符 }i\quad (L,R)\leftarrow(i,i) \] \[ \text{偶数长：中心在缝 }i|i{+}1\quad (L,R)\leftarrow(i,i+1) \]

扩展不变量

\[ \text{while } L\ge 0 \land R<n \land s[L]=s[R]: \quad L\leftarrow L-1,\; R\leftarrow R+1 \]

循环退出后，合法闭区间为

\[ [L+1,\, R-1] \] 维护全局最长的 \((\mathrm{start},\,\mathrm{len})\)。
与题 3 的差别： 题 3 用 sliding window 维护「无重复」；本题约束是左右对称， 模型是 expand around centers，不是 last-index 窗口。

这题在现实里像什么 应用场景

  • DNA / 文本里找最长「左右对称」片段（教学类比）。
  • 产品 ID、口令生成里检测对称结构。
  • 字符串算法入门：中心扩展 → Manacher 的直觉底座。
  • 与「最长回文 subsequence」对比：连续 vs 可跳，模型完全不同。

中心扩展不是炫技：回文的定义就是对称。 用中心参数化，把「所有子串」压成「所有中心」，空间常数、实现短。

多种解法 · 对比 先选武器

A. Brute force
Time \(O(n^3)\) Space \(O(1)\)

枚举所有 substring，再线性判回文。定义清晰，\(n=1000\) 偏紧。

B. Expand around centers（主推）
Time \(O(n^2)\) Space \(O(1)\)

\(2n-1\) 个中心，每个向两侧扩展；维护最长区间。 在优化什么：不枚举端点对，只枚举「对称轴」。

C. DP 区间表
Time \(O(n^2)\) Space \(O(n^2)\)

\(\mathrm{dp}[i][j]=(s[i]=s[j])\land(\mathrm{短区间或}\,dp[i+1][j-1])\)。 正确但更重；中心扩展更贴定义。

解法换到了什么丢掉了什么
Brute定义直观时间
中心扩展\(O(n^2)\)、\(O(1)\) 空间、好写要记得奇偶两种中心
DP转移清晰、可推广额外 \(O(n^2)\) 空间

为何要两种中心？ 核心

只扩「字符心」会漏掉 bb、abba 这类偶数长回文——它们的对称轴在两个字符中间的缝。

类型初值例子
奇数心 \(L=R=i\) aba 中心在 b
偶数心 \(L=i,\;R=i+1\) abba 中心在两 b 之间
实现上通常写一个 expand(l, r)：奇调用 expand(i,i)，偶调用 expand(i,i+1)。

手推两例 巩固

例 1 · babad（奇数长）

中心在 index 1 的 a：扩得 bab（或中心在 2 得 aba）。最长长度 3。

例 2 · cbbd（必须偶数心）

字符心最多单字符。缝心在两个 b 之间：\(L=1,R=2\) 扩得 bb。答案 bb。

退出后的下标

循环在「越界或不相等」时停下，此时 \(L,R\) 已各多走一步， 合法段是 \([L+1, R-1]\)，长度 \(R-L-1\)。

发散：约束一变 思维

  • 只要长度 → 同样扩展，只返回 \(\mathrm{len}\)。
  • 最长回文 subsequence → 不连续，改 DP（与本题不同）。
  • \(n\) 到 \(10^5\) 且要严格线性 → Manacher。
  • 统计所有回文个数 → 每个中心的半径可累加。

理解测 统一提交 · 作答中不公布答案

围绕中心扩展 + 奇偶中心。全对复制：#题号 · slug · 用时 · 下一步。

Q1. 本题要求的「回文子串」是？

解析：substring = 连续；多解任返一个最长。

Q2. 中心扩展主推的核心建模是？

解析：回文 = 对称；中心参数化后两侧扩展。

Q3. 为何除了「字符心」还要「缝心」？

解析：只做字符心会漏掉偶数长度回文。

Q4. 扩展循环在不相等时停下，合法区间是？

解析：while 条件失败前 \(L,R\) 多走了一步，要缩回。

Q5. \(s=\) cbbd 的一个最长回文 substring 是？（小写填写）

解析：偶数心扩出 bb；单字符更短。

Q6. 中心扩展相对暴力，主要省在？

解析：暴力常 \(O(n^3)\)；中心扩展 \(O(n^2)/O(1)\)。

Q7. 若改成「最长回文 subsequence」，主推是否仍是中心扩展？

解析：subsequence 不连续，中心扩展前提不成立。
